# JavaFX Application

A comprehensive JavaFX-based application with user authentication, file management, and load balancing features.

## Features

- User authentication and authorization
- Role-based access control
- File management system
- Load balancing with multiple scheduling algorithms
- Admin panel

## Technologies

- JavaFX
- Java
- Maven

## Setup

1. Clone the repository
2. Build with Maven: `mvn clean install`
3. Run the application
